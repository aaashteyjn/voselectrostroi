Реализованные объекты

ВОСЭЛЕКТРОСТРОЙ участвует в проектах разного масштаба — от отдельных щитов для небольших объектов до комплексной поставки НКУ для промышленных и инфраструктурных площадок. Ниже приведены примеры реализованных объектов по основным направлениям.

01

Промышленные объекты

Цеха, заводы, производственные и логистические комплексы.
02

Коммерческая недвижимость

Бизнес-центры, склады, торговые объекты.
03

Жилые комплексы и МКД

Многоквартирные дома и сопутствующая инфраструктура.
04

Инфраструктура и соцобъекты

Образование, медицина, спорт, муниципальные объекты.

01. Промышленные объекты

Проекты для промышленных предприятий включают как поставку отдельных НКУ, так и комплексные решения по электроснабжению цехов, участков и технологических линий.

Наверх

Производственный комплекс «Северный»

Тип объекта: цеха и склад готовой продукции.

  • Поставка ГРЩ и ВРУ 0,4 кВ с номиналом по вводам до 2500 А.
  • Щиты автоматики для насосных и вентиляционных установок.
  • Щиты освещения для производственных и бытовых помещений.

Логистический терминал «Восточный»

Тип объекта: складской комплекс.

  • Распределительные щиты для зон погрузки/разгрузки и стеллажных зон.
  • Щиты учёта для отдельных участков и арендаторов.
  • Щиты освещения и розеточные группы для административной части.

Пищевое производство

Тип объекта: производственные линии.

  • Шкафы управления технологическими линиями и вспомогательным оборудованием.
  • Резервирование питания ответственных участков через АВР.
  • Комплект исполнительной документации для службы эксплуатации.

02. Коммерческая недвижимость

Для объектов коммерческой недвижимости важны гибкость по арендаторам, корректный учёт и удобство обслуживания инженерной инфраструктуры.

Бизнес-центр класса B+

Тип объекта: офисное здание.

  • НКУ на вводе здания и распределительные щиты по этажам.
  • Щиты учёта для арендаторов и общедомовых нагрузок.
  • Щиты освещения для офисных и общественных зон.

Торговый центр «Городской»

Тип объекта: торговый комплекс.

  • Решение по электроснабжению торговых и технических зон.
  • Щиты учёта для магазинов и арендаторов по зонам.
  • Щиты НКУ для инженерных систем (вентиляция, кондиционирование, подсветка фасада).

Складской комплекс

Тип объекта: склады и административный блок.

  • ВРУ и распределительные щиты для складских зон.
  • Освещение стеллажей и зон погрузки/разгрузки.
  • Щиты для офисной части и вспомогательных помещений.

03. Жилые комплексы и многоквартирные дома

Для девелоперов и управляющих компаний реализуются проекты по электроснабжению и модернизации НКУ для жилых домов и сопутствующей инфраструктуры.

Жилой комплекс «Панорамный»

Тип объекта: несколько многоквартирных домов.

  • Поставка ВРУ, щитов ОДПУ и общедомовых щитов.
  • Этажные щиты и решения для коммерческих помещений на первых этажах.
  • Освещение парковок, дворовых территорий и мест общего пользования.

Модернизация МКД

Тип объекта: существующий многоквартирный дом.

  • Замена устаревших вводных щитов и этажных щитов.
  • Упорядочивание учёта и подготовка документации для энергосбыта.
  • Организация резервирования питания для ответственных нагрузок.

Дворовая и уличная инфраструктура

Тип объекта: освещение и сопутствующие системы.

  • Щиты управления наружным освещением и дворовой инфраструктурой.
  • Питание шлагбаумов, ворот, систем видеонаблюдения.
  • Решения для хозяйственных и бытовых помещений УК.

04. Инфраструктура и социальные объекты

Отдельное направление работ — проекты для объектов образования, здравоохранения, спорта, культуры и муниципальной инфраструктуры.

Школа и детский сад

Тип объекта: образовательные учреждения.

  • Электроснабжение учебных корпусов и вспомогательных помещений.
  • Щиты освещения и розеточные щиты для классов и лабораторий.
  • Решения для питания систем безопасности и связи.

Поликлиника

Тип объекта: медучреждение.

  • Вводные щиты и распределительные щиты этажей.
  • Щиты для инженерных систем (вентиляция, ИТП, лифты).
  • Резервирование питания критичных нагрузок (серверные, диагностическое оборудование).

Спортивный объект

Тип объекта: спортивный комплекс или ФОК.

  • Решения по освещению залов, арен и прилегающих территорий.
  • Щиты для инженерных систем и бытовых помещений.
  • Щиты учёта и распределения для арендаторов и услуг.
